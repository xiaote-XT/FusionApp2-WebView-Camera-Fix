# FA2 摄像头修复

一段用于 **FusionApp 2.0** 的 Lua 脚本，解决网页无法访问摄像头、无法拍照的问题。

## 它能做什么

- 自动授权网页的摄像头和麦克风权限（`getUserMedia` 不再报错）（注意，代码只能解决麦克风申请报错和摄像头访问，无法修复录音异常问题）
- 拦截网页的拍照请求，调用系统相机拍摄，并将照片返回给网页
- 附带了可测试拍照和录音的 HTML 页面

## 快速上手

1. 将 `main.lua` 中的全部代码复制到 FA2 工程里**需要启用摄像头的 WebView 页面**的 Lua 脚本中（例如放在“页面显示时”或“加载完成”事件里）。
2. 确保 App 已具有 **相机** 权限（FA2 工程属性中勾选，Android 6.0+ 可能需要动态申请）。
3. 如果你要测试，可以把项目里的 `test.html` 内容粘贴到 FA2 的网页源码中，即可看到相机预览、拍照和录音按钮，但只有相机预览是正常的。

## 注意事项

- 代码为了方便演示，**一律授予了所有权限**。如果你对安全性有要求，建议自行添加权限询问弹窗。
- 拍照部分使用了 `FileProvider` 获取 content URI，如果失败会降级为 `file://` 形式（高版本 Android 可能不可用，可按需调整）。
- 本脚本依赖 FA2 的 Luajava 环境，不能直接在其他 WebView 项目中使用。
- 一定要看注释，避免因为疏漏导致的安全问题！
